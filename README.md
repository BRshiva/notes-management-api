# Notes Management API

Production-ready REST API built with Node.js, Express, and PostgreSQL.

# Features
* JWT Authentication
* Refresh Token Mechanism
* Role-Based Access Control
* Notes CRUD
* Pagination & Search
* Swagger Documentation
* Docker Support

# Setup Instructions
* git clone <repo>
* cd notes-api
* npm install
* cp .env.example .env
* npm run dev

# Docker Run
* docker-compose up --build
